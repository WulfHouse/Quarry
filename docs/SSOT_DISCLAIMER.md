## What is the SSOT?

The **Single Source of Truth (SSOT)** for Pyrite is a **high-level master specification overview** and **hypothetical wishlist** for what the Pyrite programming language and Quarry SDK are intended to eventually become.

## Key Points

### ‚ö†Ô∏è The SSOT is Aspirational

The SSOT represents the **vision and goals** for Pyrite, not necessarily the current implementation state. It describes:

- **Intended design** - How the language should work
- **Target features** - What features are planned
- **Vision** - Where the project is heading
- **Goals** - What we're working toward
