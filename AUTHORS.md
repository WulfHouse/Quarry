# Authors

## Creator and Project Director

**Alexander Rose Wulf** (also known as **Aeowulf** and **WulfHouse**) - Creator, director, and supervisor of the Pyrite programming language and the Quarry SDK.

**Official Aliases:**

- **Aeowulf** - GitHub username

- **WulfHouse** - Organization name under which this project is published

**Pyrite** was first conceived and prototyped on December 12, 2025. **Forge** is the Pyrite compiler. **Quarry** is the official Pyrite SDK that includes Forge, build system, package manager, language server, and all development tools.

This project was conceived, directed, and supervised by Alexander Rose Wulf (Aeowulf/WulfHouse). The vast majority of code, documentation, and implementation work was generated by AI coding assistants under human supervision, review, and direction. All architectural decisions, design choices, and project direction are the work of the human supervisor.
